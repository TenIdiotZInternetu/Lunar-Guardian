# How to play
Head into the releases tab and choose a version of your liking.
Extract the content of the .zip file and run LunarGuardian

## Controls ##
* Move - WASD / Arrows
* Shoot - K / Z
* Deploy Bomb - L / X
* Focus - Shift

Navigate the menus using mouse
